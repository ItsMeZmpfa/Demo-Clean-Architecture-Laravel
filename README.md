# Demo-Clean-Architecture-Laravel
 Clean architecture with Laravel

# Development
```
php artisan serve
```
# Unit Test
```
php artisan test
```
# Description

This is an example of implementation of Clean Architecture in Laravel

It has major of 4 layers:

- Entity layer
- Repository layer
- Usecase layer
- Adapters and Ports

# Important

This application run without a view.
